# OpenNeko Homebrew Tap

Homebrew tap for the OpenNeko operator CLI.

## Install

```bash
brew install open-neko/tap/openneko
```

Formulas in this repo are published by [goreleaser](https://goreleaser.com/) from [open-neko/neko](https://github.com/open-neko/neko) on every release. Do not edit them by hand.
